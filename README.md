# How to run Weather App

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### The fire I use is
```
openweathermap.org
```

### I also use axios to retrieve data
```
npm install axios
```

### I also use boostrap
```
npm install bootstrap
```

See [Configuration Reference](https://cli.vuejs.org/config/).
